# Fixify

Paste any error or stack trace. Get a plain English explanation 
and the exact fix — instantly.

No more googling the same error twice.

## What it does
- AI explains what went wrong and why
- Gives you the exact code fix
- Saves your error history (searchable, taggable)
- Community feed — see how others fixed similar errors
- Dashboard with your debugging stats

## Stack
React · Node.js · Express · MongoDB · Gemini AI · Google OAuth

## Run Locally
```bash
git clone https://github.com/yourusername/fixify.git

cd server && npm install && npm run dev
cd client && npm install && npm run dev
```
Add your keys in server/.env — see .env.example
